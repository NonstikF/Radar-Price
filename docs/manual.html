Capítulo 00

Acerca del Sistema

¿Qué es RadarPrice y qué problemas resuelve?

RadarPrice es un sistema web de inteligencia de costos diseñado para negocios que manejan un catálogo amplio de productos. Permite controlar precios de compra y venta, gestionar el inventario físico, importar facturas XML y generar reportes de movimientos, todo desde cualquier dispositivo con navegador.

📦
Catálogo de Productos

Base de datos centralizada con SKU, UPC, costo, precio de venta y stock.

🧾
Importación XML

Carga facturas del SAT y actualiza precios y existencias automáticamente.

📍
Ubicaciones

Asigna productos a estantes y racks físicos con código de ubicación.

📊
Reportes

Historial de movimientos de existencias y resumen del inventario.

🏷️
Etiquetas

Diseña e imprime etiquetas de precio con código de barras.

👥
Multi-usuario

Gestiona roles y permisos por módulo para cada colaborador.

Requisitos de Uso

ConexiónInternet (el sistema vive en la nube)
DispositivosPC, laptop, tablet o smartphone
CámaraOpcional — para escanear códigos de barras

Capítulo 01

Acceso y Login

Cómo iniciar sesión en el sistema.

📷
Captura de pantalla: Pantalla de inicio de sesión

Pantalla de login — primer punto de entrada al sistema.

1
Abrir el sistema

Ingresa la URL del sistema en tu navegador. La pantalla de login aparecerá automáticamente.

2
Ingresar credenciales

Escribe tu usuario y contraseña en los campos correspondientes. El administrador te proporcionará estos datos.

3
Presionar "Iniciar Sesión"

El sistema te redirigirá al Panel Principal si tus credenciales son correctas.

⚠️

Si olvidas tu contraseña, contacta al administrador del sistema para restablecerla. No existe recuperación automática por correo.

Sesión y Seguridad

La sesión permanece activa por 24 horas. Para cerrar sesión, presiona el botón → (salir) en la esquina superior derecha de la pantalla.

3.1 Hub de Productos

El hub agrupa las herramientas relacionadas con el catálogo de productos: buscador de precios, entrada manual, categorías y diseñador de etiquetas.

📷
Captura de pantalla: Hub de Productos

Hub de Productos con sus sub-módulos disponibles.

3.2 Buscador de Precios

El buscador permite encontrar cualquier producto del catálogo por nombre, SKU, UPC o alias. Los resultados muestran el costo, precio de venta y existencias actuales.

📷
Captura de pantalla: Buscador de Precios

Buscador principal con tarjetas de producto, filtros y modo de selección múltiple.

Campos de búsqueda

CampoDescripción
NombreBúsqueda por nombre o parte del nombre (sin importar acentos)
SKUCódigo interno del proveedor
UPCCódigo de barras del producto
AliasNombre alternativo asignado manualmente

Tarjeta de producto

Cada producto se muestra como una tarjeta que contiene:

🏷️
Nombre y Alias

Nombre principal y nombre alternativo si existe.

💲
Precios

Costo de compra y precio de venta actual.

📊
Stock

Cantidad de unidades disponibles en inventario.

🔢
Códigos

SKU e identificador UPC/EAN visibles en la tarjeta.

Filtros disponibles

FiltroFunción
Sin precio de ventaMuestra solo productos que aún no tienen precio de venta asignado
Precio mínimo / máximoFiltra por rango de precio de venta
Stock máximoMuestra productos con existencias menores o iguales al valor indicado
Ordenar porNombre, precio, stock o fecha de última modificación

Cámara para escanear

El ícono de cámara (📷) junto al buscador activa la cámara del dispositivo para escanear un código de barras UPC directamente. El sistema busca el producto automáticamente al detectar el código.

💡

Para mejores resultados al escanear, asegúrate de tener buena iluminación y sostener el código de barras a unos 15–20 cm de la cámara.

Modo de selección múltiple

Presiona el ícono ☑ en la barra de búsqueda para activar el modo de selección múltiple. En este modo puedes:

1
Seleccionar productos

Toca cada tarjeta para marcarla. Las tarjetas seleccionadas se resaltan en índigo. También puedes usar "Seleccionar todos".

2
Acción masiva

Aparece una barra flotante abajo con el número de seleccionados y el botón "Actualizar fecha", que marca los productos como modificados hoy para que aparezcan primeros en el listado.

Editar un producto

Al tocar una tarjeta (en modo normal), se abre un modal con los detalles del producto donde puedes editar:

Campo editableDetalle
Precio de ventaSe registra en el historial de cambios de precio
SKUCódigo interno único del producto
UPCCódigo de barras del producto
AliasNombre corto alternativo
ProveedorAsignación al proveedor correspondiente
ImagenURL de imagen del producto

4.2 Importar Factura XML

El sistema acepta facturas electrónicas en formato XML del SAT (CFDI 3.3 y 4.0) y procesa automáticamente todos los productos contenidos en ella.

📷
Captura de pantalla: Importador de Facturas XML

Interfaz de carga de factura XML con resultado del procesamiento.

1
Seleccionar archivo

Arrastra el archivo .xml al área de carga o presiona "Seleccionar archivo" para buscarlo en tu dispositivo.

2
Procesamiento automático

El sistema lee cada concepto del XML e intenta identificarlo en el catálogo por SKU, UPC o nombre (con similitud aproximada).

3
Revisar resultados

El sistema muestra el listado con el estado de cada producto: ENCONTRADO NUEVO PRECIO CAMBIADO.

4
Asociar productos no encontrados

Para productos marcados como Nuevo, el sistema sugiere posibles coincidencias del catálogo. Puedes asociarlos manualmente o dejarlos como productos nuevos.

5
Actualizar precios

Presiona "Guardar cambios de precio" para confirmar las actualizaciones de costo en los productos encontrados.

Lo que hace el sistema al importar

AcciónCondición
Actualiza el costo del productoSi el precio cambió más de $0.10 respecto al anterior
Suma el stockSiempre que se importe un producto existente
Registra historial de precioSi hubo cambio de costo
Registra historial de existenciasEn cada entrada de stock
Crea proveedor automáticamenteSi el emisor del XML no existe en el sistema
Asigna proveedor al productoSi el producto no tenía proveedor asignado
⚠️

El sistema detecta facturas duplicadas por nombre de archivo. Si intentas cargar el mismo XML dos veces, lo indicará y no procesará el archivo nuevamente.

Las listas de compra permiten generar órdenes de pedido para un proveedor específico, agrupando los productos que necesitas reponer.

📷

Vista de listas activas y detalle de una lista con sus productos.

EstadoDescripción

3

4.6 Fusionar Productos Duplicados

Al importar facturas, puede ocurrir que el mismo producto quede registrado dos veces con nombres ligeramente distintos. La función de fusión permite unificarlos en un solo registro.

1
Identificar duplicado

Desde el importador de XML, en los productos marcados como "Nuevo", el sistema sugiere posibles duplicados.

2
Seleccionar "producto a conservar"

Elige cuál de los dos registros es el correcto y cuál se debe eliminar.

3
Confirmar fusión

El stock del producto eliminado se suma al conservado. El historial de ambos queda unificado.

5.1 Ubicaciones

Permite mapear el almacén físico en el sistema, definiendo estantes, racks y posiciones con un código alfanumérico. Ejemplo: R1B2 = Rack 1, Bandeja 2.

📷
Captura de pantalla: Gestión de Ubicaciones

Lista de ubicaciones con código, descripción y cantidad de productos asignados.

Crear una ubicación

1
Presionar "Nueva Ubicación"

Ingresa un código único (ej: R1B2) y una descripción opcional (ej: "Rack 1, Bandeja 2").

2
Guardar

La ubicación queda disponible para asignarle productos inmediatamente.

Vista detalle de ubicación

Al tocar una ubicación, puedes ver todos los productos asignados a ella, buscar un producto específico dentro de la ubicación, y agregar o quitar productos con su cantidad.

Usar la cámara en ubicaciones

En las siguientes secciones puedes escanear el código de barras del producto con la cámara:

SecciónFunción de la cámara
Buscar Producto (tab principal)Escanea y busca el producto en el catálogo
Modal Agregar ProductoEscanea para buscar el producto a agregar a la ubicación
Detalle de ubicaciónEscanea para buscar dentro de los productos de la ubicación

Historial de existencias

Capítulo 08

Glosario

Definición de términos usados en el sistema.

TérminoDefinición
SKUStock Keeping Unit — código interno único asignado por el proveedor para identificar un producto.
UPC / EANCódigo de barras estándar impreso en el empaque del producto.
CostoPrecio al que se compra el producto al proveedor (sin IVA).
Precio de ventaPrecio al que se vende el producto al cliente final.
StockCantidad de unidades disponibles en el inventario.
XML / CFDIArchivo de factura electrónica emitido por el SAT de México.
LoteGrupo de productos importados de una sola factura XML.
AliasNombre corto alternativo asignado manualmente a un producto para facilitar su búsqueda.
FusiónProceso de unir dos registros duplicados del mismo producto en uno solo.
UbicaciónPosición física en el almacén identificada con un código (ej: R1B2).
Historial de precioRegistro de todos los cambios de costo y precio de venta de un producto a lo largo del tiempo.
Historial de existenciasRegistro de todos los movimientos de stock (entradas, ajustes, fusiones).
Modo oscuroTema visual de la interfaz con fondo oscuro, más cómodo en ambientes con poca luz.
